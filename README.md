# CRC Card Builder

A tiny tool to create, review, and export CRC (Class–Responsibility–Collaborator) cards.

## Roadmap (MVP)
- Projects & Cards CRUD
- Issue detector (deterministic rules)
- Q&A assistant (guided, no AI)
- Export CSV + print layout
- Online help (searchable)

## Getting started
- Node 18+
- Steps will follow in README as we implement
